# Student Rating System

## Жоба туралы / О проекте
Бұл жоба Java тілінде жазылған студенттердің рейтинг жүйесі.  
Данный проект предназначен для автоматизации оценки студентов и формирования рейтинга на основе их баллов.

## Мақсаты / Цель проекта
Студенттердің бағаларын енгізу, орташа балын есептеу және рейтинг құру.  
Автоматизация процесса оценки и анализа успеваемости студентов.

## Функционал / Возможности
- Студент қосу (добавление студента)
- Студентті іздеу (поиск)
- Мәліметтерді өзгерту (редактирование)
- Студентті өшіру (удаление)
- Орташа балл есептеу (расчет среднего балла)
- Рейтинг шығару (сортировка по рейтингу)
- Әріптік баға беру (A, B, C, F)
- Статистика көрсету (анализ данных)
- Файлға сақтау және оқу (работа с файлом)

## Қолданылған технологиялар / Технологии
- Java
- Объектно-ориентированное программирование (OOP)
- Работа с файлами (File I/O)

## Жобаның құрылымы / Структура проекта
src/

## Қалай іске қосылады / Как запустить
1. Java орнатылған болуы керек  
2. Файлдарды компиляциялау:
javac src/*.java  
3. Бағдарламаны іске қосу:
java -cp src Main
